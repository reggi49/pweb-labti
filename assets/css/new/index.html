Why slideshow?

Slideshows are now one of the most popular element of webdesign. I think that about 80% new websites has some kind of it. Fading ones or scrolling. I love that first type, so I tried to make it only with CSS, because I don't know jQuery so much. And it works! Of course only in modern browsers (new IE isn't working either, bacause of not understanding @keyframes from CSS3), but I'm still thinking this slideshow is something good and useful.

That code is huge, I know that. But it's still not so huge like jQuery :)
